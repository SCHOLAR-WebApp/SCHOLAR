A Propos

Voici une description complète de l'application web de paiement des frais de scolarité.
1.INSCRIPTION

Les utilisateurs peuvent s'inscrire en entrant leur informations personnelles tellles que leurs noms,leurs numéros de téléphone,et leur mot de passe.ceci se faisant dans le cas où ils n'auraient pas encore de compte.

2.CONNEXION

Les utilisateurs enregistrés peuvent se connecter à l'application en utilisant leur nom d'utilisateur et leur mot de passe.

3.SÉLECTION DE L'ÉTABLISSEMENT

Les utilisateurs peuvent sélectionner dans une liste l'établissement où ils souhaitent éffectuer les paiement.l'application affiche les frais scolaires spécifiques pour chaque établissement ainsi que les modes de paiement valides.

4.PAIEMENT

Les utilisateurs peuvent éffectuer un paiement en choissisant les modes de paiement valides. l'application fournit un formulaire de paaiement où il choisira le montant de la/les tranche(s) à payer.

5.HISTORIQUE DES PAIEMENTS

Les utilisateurs peuvent consulter l'historique des paiements récement éffectués. il affiche la date, le montant et le statut pour chaque tranche.

6.ENREGISTRER UN NOUVEL ÉLÈVE

Les utilisateurs peuvent enregistrerle(s) élève(s) pour lequel(s) ils souhaitentrégler les frais de scolarité.

7.A PROPOS

Les utilsteurs peuvent ici se renseigner sur les différents modules de l'application.

8.SUPPORT CLIENT

Pour aider les utilisateurs et administrateurs dans les problèmes liés à l'utilisation de l'application.

9.MON COMPTE

Les utilisateurs peuvent voir les informations relatifs à leurs comptes et ils peuvent aussi modifier ou supprimer leurs informations.Et afficher les détails liés aux élèves de leurs comptes.
